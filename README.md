# Qe lsp

Language Server Protocol implementation for qe quantum chemistry software.

## Features

- Syntax highlighting
- Auto-completion
- Diagnostics
- Hover documentation

## Installation

```bash
pip install qe-lsp
```

## Usage

```bash
qe-lsp
```

## Development

```bash
git clone https://github.com/newtontech/qe-lsp.git
cd qe-lsp
pip install -e ".[dev]"
```
